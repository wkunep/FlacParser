Flac PArser

Версия 1.0

Автор: Борзенко Кирилл

Описание

Данное приложение выводит информациию основных заголовков flac файла.
Требования

    Python версии не ниже 3.4
    PyQt версии 5

Состав

    Консольная версия: main.py
    Графическая версия: gui.py
    Модули: pattern.py, flacExtractor.py, metaDataBlock.py, frame.py
    Тесты: tests.py

Для запуска тестов можно использовать coverage run tests.py (coverage3)
Покрытие приемрно 90%

Справка по запуску: main.py --help

Консольная версия

Пример запуска: python main.py -f input.flac -o output.txt

Графическая версия

Пример запуска: python main.py -f input.flac -o output.txt --gui

Подробности реализации

Если входной файл в описании содержит картинку, то эта картинка будет создана в текущей директории 'picture.png'
Есть возможность воспроизведения файла (необходим pyglet и AVbin)
